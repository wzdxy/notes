文件传输协议FTP

文件传输协议 File Transfer Protocol

互联网传输文件要解决的差异性问题

  • 存储格式
  • 目录结构命名和文件命名的规定
  • 文件存取命令
  • 访问控制方法

基本原理

  • 采用 21 端口
  • 服务器和客户机之间维持两个并行的 TCP 连接: 控制连接和数据连接, 分别由控制进程和数据传送进程管理

缺点

  • 对文件的修改需要整个读取文件, 再整个发送回去. 而 NFS (网络文件系统) 可以直接打开远程文件, 在特定位置读写数据
